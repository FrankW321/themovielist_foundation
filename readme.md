# The Movie List
A web application which lets you know about current and upcoming movies with E-mail and push-notifications

## Getting Started
```
npm install
```

```
foundation watch
```
## Built With
* [Foundation-6]https://foundation.zurb.com/) - The web framework used

## Requirements
```
npm install -g gulp bower
```
